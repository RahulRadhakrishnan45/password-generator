🔐 Password Generator

A simple and responsive Password Generator built using HTML, Tailwind CSS, and JavaScript.
It allows users to create secure passwords with customizable options.

🚀 Features
Generate random passwords instantly
Choose password length (4–30)
Include:
Uppercase letters
Lowercase letters
Numbers
Symbols
Copy password to clipboard
Responsive UI with Tailwind CSS
🛠️ Tech Stack
HTML5
Tailwind CSS
JavaScript (Vanilla)
